# AI Scientist (Catalyst)

A tool for autonomous scientific research and discovery.

![Catalyst Logo](src/assets/catalyst-small.png)

## Getting Started

1. Clone the repo
2. `git checkout stable` to use the stable branch
3. Install [prerequisites](src/README.md#prerequisites)
4. `cd src`
5. `./run.sh`
